Use of object oriented programming to create a Rectangle class and a Square class. The Square class is a subclass of Rectangle and inherit methods and attributes.

Methods:

set_width
set_height
get_area
get_perimeter
get_diagonal
get_picture: Returns a string that represents the shape using lines of "*"
get_amount_inside: Takes another shape (square or rectangle) as an argument. Returns the number of times the passed in shape could fit inside the shape (with no rotations)

Square exclusive method:
set_side
